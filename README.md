## About The Project

![Screen Shot](https://drive.google.com/file/d/1SttUtrJ3C0PnORQcISlZm4a6C1sEtqxl/view?usp=sharing)

Learning can be a challenging experience for students due to various factors such as poor time management, inadequate learning environments, unorganized content, and unclear instructions. These are issues that students face every day, hindering their academic progress.

As a former student who struggled with these challenges, I understand the frustration and lack of motivation that comes with it. That's why my team and I created ProgressWave, an innovative eLearning platform designed to revolutionize the learning experience.

Our platform is unique in its approach, as we have used extensive research to develop features that are centered around improving time management, organizing content, creating a conducive learning environment, and providing top-notch learning resources. Additionally, our platform is easily accessible to students, taking advantage of the fact that students in Nigeria have access to the internet and spend an average of 7 hours a day using their gadgets.

What sets ProgressWave apart is our commitment to empowering students beyond academic learning. We offer a wide range of skill-building opportunities in various fields, including trade, tech, business, and engineering. Moreover, we provide scholarship, sponsorship, and competition opportunities to encourage students to excel and achieve their full potential.

At ProgressWave, our goal is to make learning seamless, enjoyable, and rewarding for students. We believe that every student deserves a chance to succeed, and our platform is the stepping stone that can help them achieve their dreams.

## Getting Started

Getting started is quite easy

### Prerequisites

Here's a list of stuff you need to get started with ProgressWave

1. Download and Install Python on your PC from python.org
2. pip.
3. An internet connection.
4. Basic CLI usage knowledge

### Installation

1. Download and Install python on your machine from  [python's official website](https://python.org)

2. Clone the repo (make sure you have git installed 

```
git clone https://github.com/Tobitheprof/progresswave.git
```

3. Install pip packages

```
cd <project_directory>
pip install -r "requirements.txt"
```
4. Run the project in `CLI`

```
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
